# Hermes Memory Repository

## 简体中文介绍

此仓库用于集中存放 **Hermes** 的持久记忆（Memory）、可复用技能（Skills）以及 **MCP**（Model Context Protocol） 配置文件。通过版本控制，您可以方便地备份、同步和共享这些关键资源，提升 AI 助手的可维护性与可重复使用性。

- **Memory**：记录用户偏好、环境信息、项目约定等持久化数据；
- **Skills**：以 `Skill.md` 为格式的可复用工作流、代码片段、操作指南；
- **MCP**：模型上下文协议的配置与脚本，帮助不同模型间共享状态。

> 将相应文件放入对应目录（`memory/`, `skills/`, `mcp/`），提交后即会同步到远程仓库。

---

## English Introduction

This repository is a centralized place to store **Hermes**'s persistent memory, reusable skills, and **MCP** (Model Context Protocol) configuration files. By using Git version control, you can easily back up, synchronize, and share these essential resources, improving the maintainability and reusability of your AI assistant.

- **Memory**: Persistent data such as user preferences, environment details, and project conventions.
- **Skills**: Reusable workflows, code snippets, and guides written in `Skill.md` format.
- **MCP**: Configuration and scripts for the Model Context Protocol, enabling state sharing across different models.

> Simply place the relevant files into the appropriate directories (`memory/`, `skills/`, `mcp/`) and commit – they will be automatically synchronized to the remote repository.
